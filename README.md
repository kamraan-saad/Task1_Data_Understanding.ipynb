## Dataset Used

## Objective
The objective of this task is to understand the structure of the dataset, identify different types of data, analyze data quality, and evaluate whether the dataset is suitable for machine learning.

## Tools Used
- Python
- Pandas
- NumPy
- Jupyter Notebook

## Steps Performed
1. Loaded the Titanic dataset using Pandas.
2. Displayed first and last few records to understand dataset structure.
3. Identified numerical, categorical, ordinal, and binary features.
4. Used df.info() and df.describe() to analyze data types and statistics.
5. Checked unique values in categorical columns.
6. Identified target variable and input features.
7. Analyzed missing values and data imbalance.
